What is Django?

  • - A high-level Python Web framework
  • - Clean and pragmatic design
  • - Help developers take applications from concept to completion as quickly as possible.
  • - Django takes security seriously
  • - Exceedingly scalable

Features

  • - Object-relational mapper (ORM)

    Provides an abstraction layer (the “models”) for structuring and manipulating the data

  • - Automatic admin interface

    Generates a CRUD interface from the model's metadata.

  • - Robust template system

    Multiple Template Engine, Complex template loader, Custom backends.

  • - Performance and optimization

    Caching Framework, "Laziness", ETag, Gzip, etc.

Features

  • - Internationalization and localization.

    Full support for translation of text, formatting of dates, times and numbers, and time zones.

  • - Geographic framework

    GeoDjango intends to be a world-class geographic Web framework.

  • - API stability

    All the public APIs (everything in this documentation) will not be moved or renamed without providing backwards-compatible aliases.

  • - Secure

    ClickJacking protection, CSRF, Cryptographic signing, etc.

Brief History of Django

Django 1.3

  • - Class Based Views
  • - Static Files
  • - Logging

Django 1.4

  • - New project layout
  • - Timezone (UTC)
  • - Support for in-browser testing frameworks

Django 1.5

  • - Configurable User Model
  • - StreamingHttpResponse
  • - PostGIS 2.0 support
  • - {% verbatim %} template tag

Django 1.6

  • - Improved transactions management
  • - Persistent database connections
  • - BinaryField
  • - Malcolm Tredinnick

Django 1.7

  • - Built-in support for schema migrations
  • - App-loading refactor
  • - Improvements to Form error handling
  • - JsonResponse

Django 1.8

  • - Multiple template engines
  • - Security enhancements SecurityMiddleware
    (django-secure)
  • - New PostgreSQL specific functionality

    ArrayField, HStoreField, Range Fields, and unaccent lookup

Django 1.9

  • - Configurable Password Validation
  • - New CBV mixins (django-braces)

    AccessMixin, LoginRequiredMixin, PermissionRequiredMixin, and UserPassesTestMixin

  • - For PostGIS users

    RasterField

  • - django.contrib.postgres

    JSONField, PostgreSQL specific aggregation functions

Django 1.10

  • - Full text search for PostgreSQL

    Trigram lookup support

  • - Official support for Unicode usernames
  • - The LogEntry model now stores change messages in a JSON

Extending Django

Static files

  • - collectfast

    A Faster Collectstatic.

  • - django-pipeline

    Pipeline is an asset packaging library for Django.

  • - django-bower

    Easy way to use bower with your django project.

  • - django-storages-redux

    Support for many storages (S3, MogileFS, etc) in Django.

Images

  • - pillow

    Python Imaging Library.

  • - easy-thumbnails

    A powerful, yet easy to implement thumbnailing appl.

  • - django-cleanup

    Automatically deletes files for FileField, ImageField, and subclasses.

User

  • - django-allauth

    Addresses authentication, registration, account management as well as 3rd party (social) account authentication.

  • - django-registration

    A simple, extensible user-registration app for Django.

  • - django-registration-defaults

    15 templates required to implement the workflow of registration, login/logout, and password changing and resetting.

  • - django-password-reset

    Class-based views for password reset.

Forms

  • - django-forms-bootstrap
  • - django-multiselectfield
  • - django_select2
  • - django-countries
  • - django-money
  • - django-form-utils

    Form enhancements for Django (fieldset and row-attrs support, and more).

  • - django-widget-tweaks

    Altering CSS classes and HTML attributes is supported.

API

  • - djangorestframework

    Powerful and flexible toolkit for building Web APIs.

  • - django-rest-swagger

    Swagger Documentation Generator for Django REST Framework.

Background Task

  • - django-celery

    Celery is an asynchronous task queue/job queue.

  • - seacucumber

    Sea Cucumber routes email through Amazon Web Services' excellent Simple Email Service (SES).

Miscellaneous

  • - django-extensions

    A collection of custom extensions for the Django

  • - django-uuslug

    Returns a unique (& unicode) slugs in Django.

  • - django-autoslug

    An automated slug field for Django.

  • - django-contrib-comments

    This framework can be used to attach comments to any model.

  • - django-tables2

    Automatic table generation based on a Django model. Supports jQuery DataTables.
